# Vending Management System

This project is a digital management system designed to optimize and automate the daily operations of a vending machine business.

## Overview

The goal of this application is to replace manual processes (paper notes, spreadsheets, and informal tracking) with a centralized system that improves efficiency, organization, and decision-making.

## Key Features

### Route & Task Planning
- Weekly route management for restocking machines
- Integrated calendar to schedule and organize tasks
- Automatic task generation based on machine restocking frequency

### Inventory Management
- Track products added to each vending machine
- Automatic stock updates when machines are refilled
- Warehouse stock control with real-time updates

### Purchase & Expense Tracking
- Register product purchases
- Automatic stock increase after purchases
- Expense tracking for better financial visibility

### Business Insights
- Monitor revenue and performance
- Track profits and operational efficiency

### Shopping List Automation
- Generate purchase lists based on stock levels and needs

## Purpose

This system was developed as a real-world solution for a vending machine business, aiming to:
- Reduce manual workload
- Improve operational efficiency
- Centralize all business data in one place
- Enable better decision-making

## Technologies Used

- Web Development: HTML, CSS, JavaScript, PHP
- Databases: MySQL
- Development Environment: XAMPP
- Tools: Git
